Proyecto GymRats
Aplicación basada en Flask para gestión de usuarios y productos en una tienda de suplementos y accesorios. Este archivo indica cómo configurarla y ejecutarla en tu máquina local.

Requisitos
- Python 3.11+
- pip (gestor de paquetes de Python)

Instalación
Sigue estos pasos para configurar el entorno virtual y dependencias:

1) Crear un entorno virtual:
   python -m venv venv

2) Activar el entorno virtual (Windows):
   .\venv\Scripts\activate

3) Instalar dependencias:
   pip install -r requirements.txt
